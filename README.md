# GUI_GRPO

GRPO 强化学习框架与相关 GUI Agent 代码。

## 目录结构

```
GUI_GRPO/
├── EasyR1/          # 多模态 RL 训练（GRPO），含 Android 数字游戏定制
└── Mind2Web/        # 网页 Agent 数据集与基线代码
```

## 快速开始（EasyR1）

```bash
cd EasyR1
pip install -e .
bash examples/qwen2_5_vl_3b_android_gui_grpo.sh
```

详见 `EasyR1/examples/android_gui_cookbook/README.md`。
